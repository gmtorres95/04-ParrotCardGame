## How to run

### Install the application

```bash
# Clone this repository
$ git clone https://github.com/gmtorres95/ParrotCardGame

# Install the dependencies
$ npm i
```

### Run the application

```bash
$ npm start
```

The server will automatically start at localhost:3000/
